# Excel

## Запуск
```bash
# склонировать проект
$ git clone https://github.com/camward/Excel.git

# установить зависимости
$ npm i

# запустить приложение
$ npm start
```

## Тестирование
```bash
# запустить тесты
$ npm run test
```

## Примечание
- для выбора нескольких ячеек используйте клавишу `Shift`
- для расчёта формул используйте `=` в поле формулы. например, `=3+2`
